Enkel sida för att kunna söka efter GitHub users, skapat med JavaScript, JQuery, JSON och Ajax. Stylad med simpel css/bootstrap

* Öppna Index.html 
* Ange profilen ni vill söka efter
* Vid träff kommer en detaljerad lista upp 
* Det kommer att visas användarens repos som man kan klicka sig vidare till
* En användare utan Repos kommer få upp ett meddelande där det står "Inga repos!"
